# VK-Practice - VK Quiz

Интерактивное приложение для создания и прохождения квизов в реальном времени.

## 📋 Описание

VK Quiz - это веб-приложение для организации командного квиза. Пользователи могут создавать комнаты для квизов, приглашать других участников через код комнаты и совместно проходить вопросы в реальном времени.

### Основные возможности:

- 🎯 **Создание квизов** — удобный интерфейс для создания вопросов с вариантами ответов
- 🔗 **Присоединение к комнате** — присоединение к квизу по уникальному коду комнаты
- 👥 **Мультиплеер режим** — реальное время для участников
- 📊 **Результаты** — просмотр результатов после прохождения квиза
- 🎨 **Стильный интерфейс** — минималистичный дизайн с использованием компонентов VKUI

## 🚀 Быстрый старт (Docker)

Нужны только **Docker** и **Docker Compose**.

```bash
# Из корня репозитория VK-Practice
make up
```

Откройте в браузере: **http://localhost:3000**  
API: **http://localhost:8050** (например, `GET /api/health`)

Остановка:

```bash
make down
```

Логи всех сервисов: `make logs`

## 🚀 Локальная разработка (без Docker)

### Требования
- Node.js 18+ (рекомендуется 20+)
- npm или yarn
- PostgreSQL 16+ (или поднимите только БД: `docker compose up -d postgres`)

### Установка

```bash
# Перейти в директорию проекта
cd frontend/vkquiz

# Установить зависимости
npm install
# или
yarn install
```

### Разработка

```bash
# Запустить development сервер
npm run dev
# или
yarn dev
```

Приложение будет доступно по адресу: **http://localhost:3000**

### Production

```bash
# Собрать приложение
npm run build

# Запустить production версию
npm start
```

### Проверка кода

```bash
# Запустить ESLint
npm run lint
```
